# 演示录制清单

建议录制 3 到 5 分钟，重点展示“完整循环”而不是只展示战斗画面。

## 录制前准备

- 打开 `Game/Assets/Scenes/Main.unity`。
- 确认游戏窗口比例为 16:9，例如 1280x720 或 1920x1080。
- 确认主菜单显示为全屏页面，UI 没有文字重叠。
- 可提前给局外金币加一点存档，也可以从 0 金币开始展示。

## 推荐录制顺序

1. 主菜单
   - 展示标题、局外金币、永久强化项。
   - 简单点击一个可升级天赋，如果金币不足则说明这是局外养成入口。

2. 进入战斗
   - 展示 WASD 移动、鼠标准心、虚线瞄准线。
   - 展示自动武器攻击和敌人追击。

3. 升级选择
   - 拾取经验水晶，展示升级三选一。
   - 指出主动/被动槽位限制和技能图标。

4. 武器效果
   - 展示手枪、霰弹枪、电磁手套、黑洞、光刃、激光中至少三种。
   - 如果能触发超进化，重点展示超进化前后的特效变化。

5. 补给和场景交互
   - 打碎补给箱。
   - 展示金币、烤鸡、磁铁或炸弹。

6. 暂停菜单
   - 按 `ESC` 展示主动武器槽、被动技能槽、作战状态和超进化搭配。

7. 结算
   - 主动重开或让角色死亡，展示结算面板。
   - 展示局内金币转换为局外金币。

## 讲解词参考

```text
这是我的 Unity/Tuanjie 2D Roguelite 射击游戏《僵尸狂潮：超载重启》。
玩家通过 WASD 移动，鼠标控制瞄准方向，武器会自动攻击。
局内通过击杀敌人收集经验升级，每次升级可以选择主动武器或被动技能。
游戏限制 3 个主动武器和 3 个被动技能，武器满级并搭配指定被动后可以触发超进化。
局外金币可以在主菜单中升级永久天赋，让下一局开局更强。
```
