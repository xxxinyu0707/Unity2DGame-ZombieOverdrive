# Game Brief Draft

## Candidate Concept

Working title: `Campus Dash`

Genre: 2D top-down single-player arcade collector

Player fantasy: rush around campus before the deadline, collect assignment fragments, dodge distractions, and submit the final project on time.

## Core Loop

1. Move around the map.
2. Collect assignment fragments.
3. Avoid or outrun moving obstacles.
4. Reach the submission point before time runs out.
5. Score is based on collected fragments and remaining time.

## Why This Scope Works

- Single scene can demonstrate a complete game.
- Mechanics are easy to explain in a presentation.
- Can be built with simple sprites and Unity physics.
- Polish can be added gradually: UI, sound, particles, multiple levels, enemy paths.

## Minimum Feature Set

- Player movement
- Camera follow
- Collectibles
- Timer and score UI
- Win and lose states
- Restart button
- Windows build

## Stretch Goals

- Patrol enemies
- Speed boost item
- Level selection
- Sound effects and background music
- Main menu and pause menu
- Save best score locally
