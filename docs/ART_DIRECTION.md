# Art Direction

## Style

`Zombie Overdrive` uses a compact top-down pixel style. The current asset pass is intentionally project-generated instead of imported from external packs, so it has no third-party art dependency.

## Generated Asset Set

- Player survivor sprite
- Walker, Runner, Spitter, Tanker enemy sprites
- Mid boss and final boss sprites
- Bullet, acid projectile, singularity orb, and experience crystal sprites
- Looping cracked ground tile

## Pipeline

Pixel assets are generated by `Game/Assets/Editor/ZombieOverdriveSceneBuilder.cs` into `Game/Assets/Art/Generated/`.

To regenerate the current art and scene in the editor:

1. Open the project in Tuanjie/Unity.
2. Run `Zombie Overdrive > Build Prototype Scene`.
3. Open `Assets/Scenes/Main.unity` and press Play.

The generated textures use point filtering, disabled mipmaps, and transparent alpha so they stay crisp in the top-down camera.
